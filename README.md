# Related Papers

Found recent related papers for me.

## Prerequisite

- Python 3.7
- pipenv

## Installation

- `pipenv install`
- `pipenv run build`

## Execution

### Training

`pipenv run train`

### Testing

`pipenv run start`

### Collect some data for manual annotation

`pipenv run collect`

### Statistics

`pipenv run stats`
